## ✨ ویژگی‌ها

### قابلیت‌های اصلی
- 🔄 **انتقال خودکار پیام** از تلگرام به بله
- 👥 **مدیریت چند ادمین** با سطوح دسترسی مختلف
- 🎯 **فیلترهای پیشرفته** برای کنترل دقیق پیام‌ها
- 📎 **پشتیبانی از انواع مدیا** (عکس، ویدیو، صدا، فایل)
- 🔐 **سیستم امنیتی کامل** با قابلیت تعریف ادمین اصلی

### قابلیت‌های مدیریتی
- 👑 **ادمین اصلی** با دسترسی کامل
- 👤 **ادمین معمولی** با دسترسی محدود
- ➕ **افزودن/حذف ادمین** (فقط توسط ادمین اصلی)
- 📋 **مشاهده لیست ادمین‌ها**

### قابلیت‌های فیلتر
- 🎚️ **فیلتر بر اساس نوع چت** (پیوی، گروه، کانال، ربات)
- 📝 **افزودن دستی چت‌ها** به لیست فیلتر
- 🌍 **حالت "همه"** برای دریافت تمام پیام‌ها
- ⚙️ **تنظیمات لحظه‌ای** بدون نیاز به ریستارت

### قابلیت‌های تعاملی
- 📥 **دریافت پیام** از چت‌های تلگرام
- 📤 **ارسال پیام** به چت‌های تلگرام
- 🔍 **دریافت مشخصات** کاربران و کانال‌ها
- 📊 **صفحه‌بندی خودکار** لیست چت‌ها

## 🚀 نصب و راه‌اندازی

### 1. دریافت کد پروژه

```bash
git clone https://github.com/Scorpian-my/BaleTel.git
cd BaleTel

wget https://github.com/Scorpian-my/BaleTel/archive/refs/heads/main.zip
unzip main.zip
cd BaleTel-main
```
### 2. ایجاد محیط مجازی (توصیه شده)
```bash
# Windows
python -m venv venv
venv\Scripts\activate

# Linux/Mac
python3 -m venv venv
source venv/bin/activate
```

### 3. نصب کتابخانه‌ها

```bash
pip install -r requirements.txt
```
### 4. دریافت اطلاعات مورد نیاز

#### 📱 دریافت اطلاعات تلگرام
1. به سایت [my.telegram.org](https://my.telegram.org) بروید
2. وارد حساب کاربری خود شوید
3. به بخش "API development tools" بروید
4. یک اپلیکیشن جدید بسازید
5. مقادیر `API_ID` و `API_HASH` را کپی کنید

#### 🤖 دریافت توکن ربات بله
1. به ربات `@BotFatherBale` در بله پیام دهید
2. دستور `/newbot` را ارسال کنید
3. نام و یوزرنیم ربات را انتخاب کنید
4. توکن دریافتی را ذخیره کنید

#### 📩 دریافت آیدی عددی در بله
1. به ربات `@UserInfoBot` در بله پیام دهید
2. ربات آیدی عددی شما را ارسال می‌کند
---
### 5. تنظیم فایل محیطی
فایل .env را ایجاد و یا ویرایش کنید:

```env
API_ID=12345678
API_HASH="your_api_hash_here"
SESSION_NAME="telegram_session"

TOKEN="your_bale_bot_token_here"
ADMIN_BALE=123456789
```
⚠️ نکات امنیتی:

هرگز فایل .env را در Git commit نکنید

از .gitignore برای محافظت استفاده کنید

توکن‌ها را با کسی به اشتراک نگذارید
---
### 6. اجرای ربات
```bash
python main.py
```
در اولین اجرا، از شما اطلاعات لاگین تلگرام خواسته می‌شود:

### 📁 ساختار پروژه
```
telegram_bale_bot/
│
├── config/                         # پوشه تنظیمات
│   └── config.py                 # تنظیمات و کانفیگ (دریافت از .env)
│
├── filters/                        # پوشه فیلترها
│   └── filters.py              # تنظیمات فیلتر خودکار پیام‌ها
│
├── handler/                        # پوشه هندلرها
│   └── handler.py         # هندلر پیام‌های تلگرام
│
├── main.py                          # فایل اصلی و اجرایی ربات
├── requirements.txt                 # کتابخانه‌های مورد نیاز
├── .env                            # متغیرهای محیطی (توکن‌ها و تنظیمات)
├── LICENSE                         # مجوز پروژه
├── README.md                       # مستندات پروژه
└── telegram_session.session        # فایل سشن تلگرام (پس از اجرا ساخته می‌شود)
```
